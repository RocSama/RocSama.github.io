HDU 2044.一只小蜜蜂

发表于 2016-07-21   |   分类于 算法练习

题目

Problem ## Description
有一只经过训练的蜜蜂只能爬向右侧相邻的蜂房，不能反向爬行。请编程计算蜜蜂从蜂房a爬到蜂房b的可能路线数。
其中，蜂房的结构如下所示。

Input

输入数据的第一行是一个整数N,表示测试实例的个数，然后是N 行数据，每行包含两个整数a和b(0<a<b<50)。

Output

对于每个测试实例，请输出蜜蜂从蜂房a爬到蜂房b的可能路线数，每个实例的输出占一行。

Sample Input

2
1 2
3 6

Sample Output

1
3

题解

这题HDU给归类到一个递归专题里了，然后我就冲着递归写，结果一直超时...讨论区似乎很多人也是递归写不通过
然后又是找规律，发现只要是蜂窝之间差值相等的话，路线种数是一样的，比如从1到3，或2到5，或3到6...这点从图上可以看出来，而且结果满足斐波那契数列...这就好说了~
注意斐波那契的数组要用long long，不然最后会溢出~
